## Note

| Speaking | Writing              |
| -------- | -------------------- |
| Also     | Moreover             |
| Anyway   | Nevertheless         |
| I think  | In my opinion (~IMO) |
| So       | Therefore            |
| And      | In addition          |
| But      | However              |
